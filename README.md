# LMS-SCH
# 🚀 LMS-School (Enterprise Learning Management System)

LMS-School adalah platform Learning Management System (LMS) modern yang dirancang untuk kebutuhan sekolah, institusi pendidikan, dan EdTech skala nasional.

Dibangun dengan pendekatan **Modular Monolith + API-first Architecture**, sistem ini scalable, secure, dan production-ready.

---

# 🧠 Tech Stack

## Backend
- Laravel 12
- PHP >= 8.5
- MySQL
- Redis (Cache & Queue)
- Laravel Sanctum (Authentication)
- Spatie Permission (RBAC)
- Laravel Filesystem (S3 Ready)

## Frontend
- Next.js (App Router)
- TypeScript
- TailwindCSS
- ShadCN UI
- React Query
- Zustand
- React Hook Form + Zod

---

# 🏗️ Architecture

- Modular Monolith
- Clean Architecture:
  - Controllers
  - Services
  - Repositories
  - DTO
  - Policies
